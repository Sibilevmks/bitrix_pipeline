# bitrix_pipeline
Основной пайплайн в системе автоматизации брокерского Битрикс24 облако. Скачивает из битрикса данные, так же ищет на Gosbase закупки и победителей и находит их среди клиентов и лидов компании.
Конфиги и функции обезличены. Для запуска требуется данные Битрикс24 (поля, стадии и т.д.) Выложен, как пример работы. Пайплайн запускается каждые 15 минут по крону.
Скрипты-
sync_staff.py - Синхронизирует сотрудников компании и таблицу с сотрудниками в БД.
mpb_deals.py - Загружает в БД сделки, звонки, количество лидов.
company_dash - Заполняет БД для статистики по клиентам.
gosbase_win.py - Ищет на Gosbase.ru тендеры, берет от туда победителей закупок и ищет их в лидах и компаниях Битрикса. Если находит создает лид назначая менеджера лида/компании ответсвенным, а руководителя добавляет наблюдателем.
